# vanilla-countdown
A countdown plugin made with vanillaJS

parameters: options {object}, element {object(DOM_NODE)}

use example:

<pre><code>
var counter = new vanillaCountdown({
		year: 2015,
		month: 11,
		day: 27,
		hour: 23,
		minute: 59,
		second: 59
	}, element);
	
	counter.init();
</code></pre>
